# 🤖 AI Automation Agent

An AI-powered automation agent that understands user intent and performs tasks using natural language.

---

## 🚀 Features

- 💬 Chat-based AI assistant  
- 📄 Text summarization  
- 🧠 Concept explanation  
- 📧 Automated email sending  
- ⚡ Autonomous decision-making (auto-detects tasks)

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- OpenAI API / OpenRouter  
- SMTP (Email automation)

---

## ▶️ How to Run Locally

1. Install dependencies:
2. pip install -r requirements.txt

2. Run the app:
streamlit run app.py

---

## 🌍 Usage

Examples:

- "Explain artificial intelligence"
- "Summarize this paragraph..."
- "Send email to xyz@gmail.com saying I am sick"
